Bot Commands

Once configured, the following commands will be available in your bot:

Command Description
/start Start the bot and verify admin access
/add [number] Add a number to test123 (e.g., /add 12025551234)
/remove [number] Remove a number from test123
/numbers List all numbers in test123
/otp [number] Get latest OTP for a specific number
/stats View OTP statistics
Instructions
  1. Create a Bot
    Message @BotFather on Telegram and follow the instructions to create a new bot.
  2. Get the Token
    Copy the bot token provided by BotFather.
  3. Create a Channel
    Create a Telegram channel where OTP messages will be forwarded.
  4. Add Bot as Admin
    Add your bot to the channel and make it an admin with posting permissions.
  5. Get Channel ID
    Forward a message from your channel to @ChannelIDBot to get the channel ID.
  6. Get Your Chat ID
    Message @userinfobot to get your personal Telegram Chat ID.
Security
  • All messages are automatically masked:
  • Phone: Last 7 digits replaced with X (e.g., 1202XXX-XXXX)
  • CLI: Last 6 characters replaced with X (e.g., WhatsXX)
