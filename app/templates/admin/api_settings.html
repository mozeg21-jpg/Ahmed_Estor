API Documentation
Send SMS
POST /api/sms/send
Headers: X-API-Token: <your_token>
Body: { number, destination, cli, message }
Bulk SMS
POST /api/sms/send-bulk
Headers: X-API-Token: <your_token>
Body: { number, destinations: [], cli, message }
Receive SMS Webhook
POST /api/sms/receive
Headers: X-Webhook-Secret: <your_secret>
Body: { number, from, cli, message }
Get Numbers
GET /api/sms/numbers
Headers: X-API-Token: <your_token>
SMS Inbox
GET /api/sms/inbox
Headers: X-API-Token: <your_token>
